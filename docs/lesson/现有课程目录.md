# 现有课程目录

本文档是 [`src/lessons/data.ts`](../../src/lessons/data.ts) 的可读镜像，记录 Git Learn 当前全部关卡与步骤。**修改课程时以代码为准**，本文档供查阅与规划参考。

- **统计**：13 个 World、57 个 Step
- **引擎**：World 1–12 为 Sim（模拟）；World 13 为 Real（真实 Git）
- **如何使用网站**：见 [使用指南](../使用指南.md)
- **场景化重组规划**：见 [场景化课程规划](./场景化课程规划.md)

## 类型说明

| 字段 | 含义 |
|------|------|
| `mode` | `sim` = 轻量模拟引擎；`real` = 浏览器内 isomorphic-git |

## World 9 — 撤销基础

- **id**: `world9`
- **模式**: sim
- **种子**: `empty`
- **描述**: 学习 restore 与 reset 的温和撤销方式。

| 步骤 id | 标题 | 指令摘要 | commandHint | 风险 | 安全提示 |
|---------|------|----------|-------------|------|----------|
| w9-init-commit | 建立基线提交 | 依次完成 git init -> git add . -> git commit -m "base"。 | `git commit -m "base"` | basic | |

## World 10 — 风险与回滚

- **id**: `world10`
- **模式**: sim
- **种子**: `empty`
- **描述**: 学习 amend、revert 与高风险 reset --hard。

| 步骤 id | 标题 | 指令摘要 | commandHint | 风险 | 安全提示 |
|---------|------|----------|-------------|------|----------|
| w10-setup | 准备提交 | 完成 init -> add . -> commit -m "release"。 | `git commit -m "release"` | basic | |
| w10-amend | 修改最近提交 | 输入 git commit --amend -m "release v2"。 | `git commit --amend -m "release v2"` | advanced | 仅适用于未 push 的最后一次提交。 |
| w10-revert | 反向提交回滚 | 输入 git revert HEAD（团队协作推荐方式）。 | `git revert HEAD` | advanced | 已 push 历史优先使用 revert，而不是 reset --hard。 |
| w10-reset-hard | 硬回退（危险） | 输入 git reset --hard HEAD~1，体验高风险回退。 | `git reset --hard HEAD~1` | danger | 危险：会丢弃提交与改动。公共分支请改用 git revert。 |

## 附录 A：Git 命令索引

按类别汇总全部 13 关涉及的 Git 命令（含常用参数）：

| 类别 | 命令 |
|------|------|
| 仓库初始化 | `git init` |
| 暂存 | `git add .`、`git add <file>`、`git add .gitignore` |
| 提交 | `git commit -m`、`git commit -am`、`git commit --amend` |
| 历史与状态 | `git log`、`git log --oneline`、`git log <range>`、`git status`、`git diff`、`git diff --staged` |
| 分支 | `git branch`、`git branch -M`、`git branch <name>`、`git switch -c`、`git checkout -b` |
| 合并与变基 | `git merge`、`git rebase`、`git rebase origin/main` |
| 远程 | `git remote add`、`git remote -v`、`git remote set-url`、`git clone`、`git fetch`、`git pull`、`git pull --rebase`、`git push -u` |
| 暂存现场 | `git stash`、`git stash list`、`git stash pop` |
| 撤销与回滚 | `git restore`、`git restore --staged`、`git reset HEAD`、`git reset --soft`、`git reset --hard`、`git revert` |
| 忽略规则 | `.gitignore`（通过 add）、`git rm -r --cached` |
