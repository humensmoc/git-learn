# 对话主题记录

本 Skill 专用于 **Notebook 项目**，记录**当前会话**中每一轮的独立输入与输出：用户说了什么、AI 回复了什么、具体时间。每轮单独追加到 `对话日志/YYYY-MM-DD.md`，不汇总跨会话。

## 触发方式

- **每次必执行**：用户每发一条消息，回复完成后都要执行一次记录，无例外
- 不区分对话是否结束，不询问，直接记录

## 执行步骤

1. **确定时间**：记录当前具体时间（HH:MM 或 HH:MM:SS）
2. **总结用户消息**：用 1–2 句话概括用户本轮说了什么
3. **总结 AI 回复**：用 1–2 句话概括本轮的回复内容
4. **确定文件路径**：`对话日志/YYYY-MM-DD.md`（按当日日期）
5. **追加记录**：在文件末尾按格式追加

## 输出格式

每条记录对应一轮对话（用户一条消息 + AI 一条回复）：

```markdown
### 回合 N | HH:MM
- **用户**：[用户消息的大致总结]
- **回复**：[AI 输出的大致总结]
---
```

**回合编号 N**：当日该文件中的第几条记录（从 1 递增）。

**当日文件结构**（若新建）：

```markdown
# YYYY-MM-DD 对话日志

### 回合 1 | HH:MM
- **用户**：...
- **回复**：...
---
```

## 文件路径

- **文件夹**：`对话日志/`（项目根目录下）
- **文件名**：`YYYY-MM-DD.md`，每天一个文件
- 若文件夹不存在，则创建
